Мобильное приложение

Лучшее мобильное приложение для вашей защиты!

Raptor - мобильное приложение для работы с охранной сигнализацией.

Где бы вы не находились, с помощью приложения можно взять объект под охрану или снять его с охраны используя персональный пароль.
С помощью push уведомлений вы можете видеть, что происходит на объекте. Кем из пользователей снят или поставлен объект на охрану. Так же вы увидите сообщение о Тревоге.
Приложение «Raptor» доступно только для клиентов охранного предприятия. Для работы приложения необходимо подключение к интернету.
